Instructions
  • Encrypt: Select a file, choose "Encrypt File", enter a key, and click "Process File"
  • Decrypt: Select an encrypted file, choose "Decrypt File", enter the same key used for encryption, and click "Process File"
  • Key: Use the same key for encryption and decryption. Keep it secure!
